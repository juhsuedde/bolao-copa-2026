# ⚽ Bolão Copa 2026

Aplicação web (PWA) para bolão entre amigos durante a Copa do Mundo de 2026.

## 🚀 Objetivo

Permitir que usuários:

- Façam palpites em jogos
- Acompanhem ranking em tempo real
- Participem de apostas com sistema de pontuação

## 🛠️ Tecnologias (planejadas)

- Frontend: React + TypeScript + Tailwind
- Backend: Supabase (PostgreSQL + Auth + Realtime)
- API de dados: WC 2026 API / API-Football

## 📱 Plataforma

Aplicação web otimizada para iPhone (PWA)

## 🎨 Protótipo atual

Protótipo em HTML/CSS simulando as telas principais:

- Home (ranking)
- Jogos
- Especiais
- Ranking detalhado
